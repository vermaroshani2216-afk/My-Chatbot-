
# 🌱 My Chatbot

## 📖 Project Overview

My Chatbot is an AI-powered farming support chatbot developed during my internship. It is designed to assist farmers by providing useful information related to modern farming practices, including irrigation, fertigation, crop management, and agricultural support.

The chatbot is built using Python, Gradio, and Hugging Face Spaces to provide an easy-to-use interface where farmers can ask farming-related questions and receive quick guidance.

---

## 🎯 Project Objectives

- Support farmers with farming-related queries.
- Provide guidance on irrigation and fertigation.
- Improve access to agricultural information.
- Demonstrate the use of AI in agriculture.
- Create a simple and user-friendly chatbot interface.

---

## ✨ Features

- 🌾 Answers farming-related questions.
- 💧 Provides irrigation guidance.
- 🌱 Explains fertigation concepts.
- 🤖 AI-powered chatbot interface.
- 💻 Simple and easy-to-use design.
- ⚡ Fast response through Gradio interface.

## 🚀 How to Run the Project

1. Clone this repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python app.py
```

4. Open the Gradio interface in your browser.
